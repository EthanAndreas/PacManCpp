## Usage
### Controls
| Key                 | Action     |
| ------------------- | ---------- |
| `UpArrow` or  `Z`   | Move up    |
| `DownArrow` or `S`  | Move down  |
| `LeftArrow` or `Q`  | Move left  |
| `RightArrow` or `D` | Move right |
| `Space`             | Pause      |
| `Escape`            | Quit       |
